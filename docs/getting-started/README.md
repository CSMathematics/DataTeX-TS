# Getting Started

Welcome to the **DataTex v2** documentation. This section will guide you through installing the application and setting up your environment for the best experience.

## What you'll need

- A computer running **Windows**, **macOS**, or **Linux**.
- A standard **LaTeX distribution** installed (e.g., TeX Live, MiKTeX) to compile your documents.
- (Optional) **Git** if you plan to contribute or build from source.
