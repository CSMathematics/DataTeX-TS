# Visual Wizards

LaTeX is powerful, but its syntax can be verbose and hard to remember. DataTex includes **Visual Wizards** to generate complex code for you.

## Table Wizard

Create professional tables without wrestling with `&` and `\\`.

1.  **Launch**: Click the Table icon in the toolbar.
2.  **Edit**:
    - Select number of rows and columns.
    - Click cells to merge them.
    - Set alignment (Left, Center, Right) visually.
3.  **Insert**: The wizard generates the `\begin{tabular}...` code and inserts it at your cursor.

## TikZ & PGFPlots Wizard

Draw vector graphics using a GUI.

- **Shapes**: Drag and drop rectangles, circles, and lines.
- **Plots**: Enter a mathematical function (e.g., `y = sin(x)`) to generate a plot.
- **Styling**: Adjust colors, line thickness, and labels using the properties panel.

## Preamble Wizard

Manage your document settings globally.

- **Packages**: Checkbox list to enable common packages (`amsmath`, `geometry`, etc.).
- **Page Layout**: Set margins and paper size visually.
- **Fonts**: Choose your document fonts without remembering package names.
