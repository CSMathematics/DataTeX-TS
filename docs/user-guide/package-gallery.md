# Package Gallery

The **Package Gallery** is your built-in comprehensive catalog of LaTeX packages.

## Features

- **Browse**: View packages by category (Graphics, Math, Fonts, etc.).
- **Search**: Find packages by name or keyword.
- **Details**: Click a package to see:
  - Description and purpose.
  - Common usage examples.
  - Link to official CTAN documentation.

## Using a Package

1.  Open the **Packages** view from the Activity Bar.
2.  Find the package you need (e.g., `fancyhdr`).
3.  Click **Add to Preamble**.
    - DataTex adds `\usepackage{fancyhdr}` to your active document's preamble automatically.
