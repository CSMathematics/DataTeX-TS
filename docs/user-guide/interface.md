# Interface Overview

DataTex features a modern, clean interface inspired by popular code editors like VS Code, but optimized for LaTeX.

## The Layout

### 1. Activity Bar (Leftmost)

Allows you to switch between primary views:

- **Files**: Explorer for your Collections and Documents.
- **Search**: Global search across all your databases.
- **Packages**: Access the Package Gallery.
- **Settings**: Configure editor preferences and themes.

### 2. Sidebar

Displays contextual information based on the active view. For example, in **Files** view, it shows the tree of your Collections and the files within them.

### 3. Editor Area (Center)

The main workspace where you write your LaTeX code.

- Supports **tabs** for multiple open files.
- Provides **syntax highlighting** and **autocompletion**.
- **Minimap** for quick navigation through long documents.

### 4. Right Panel

Contains auxiliary tools:

- **PDF Viewer**: Real-time preview of your compiled document.
- **Structure**: Outline of your current document (Sections, Labels).
- **Context Help**: Shows documentation for commands under the cursor.

### 5. Status Bar (Bottom)

Shows current cursor position, file encoding, and compilation status.
