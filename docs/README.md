# DataTex v2 - Modern LaTeX IDE

**DataTex v2** is a next-generation LaTeX editor built for speed, usability, and modern workflows. Powered by **Tauri**, **React**, and **Rust**, it goes beyond standard text editing by offering a suite of visual tools that simplify complex LaTeX tasks.
