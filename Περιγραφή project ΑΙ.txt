Φτιάχνω ένα project που είναι η 2η έκδοση του DataTex : https://github.com/CSMathematics/DataTex
το οποίο είναι ένα πρόγραμμα δημιπουργίας και διαχείρισης βάσεων δεδομένων αρχείων Latex σε βάσεις .db με sqlite3.
Είναι επίσης ένα πλήρες IDE για Latex με πολλές λειτουργίες όπως wizards για δημιουργία πακέτων και κλάσεων Latex, preamble, πινάκων διαγραμμάτων tikz, pgfplots και άλλα.
Θέλω να είναι απλό για το χρήστη δηλαδή να έχει όσο το δυνατόν ευκολότερο ui και ux χωρίς πολλά popup παράθυρα. Να μοιάζει με το vscode δηλαδή ένα μοντέρνο Latex IDE
Xρησιμοποιώ tauri και mantine-ui όποτε το backend θα είναι σε Rust και το frontend σε Typescript/Javascript.


Εχω μια ιδέα για την κατασκευή ενός γενικού package manager για όλα τα πακέτα του latex όπου ο χρήστης θα μπορεί να βλέπει για κάθε πακέτο τις δυνατότητες, εντολές και options που έχει και να δημιουργεί γρήγορα περιεχόμενο για τα latex έγγραφά του. Πχ πέρα από την εισαγωγή του listings στο preamble, να υπάρχει και ένας wizard για όλα τα σημαντικά πακέτα του latex όπου όλες οι επιλογές του πακέτου να εισάγονται διαδραστικά μέσα στο έγγραφο και όχι πλέον στο preamble


Ωραία μια χαρά σε πρώτη φάση. Το φαντάζομαι ως εξής: Θα εμφανίζονται τα πακέτα σε μια λίστα ή treeview ή ότι ταιριάζει καλύτερα και για κάθε πακέτο επιλέγοντας το θα εμφανίζεται ένα παράθυρο με τις επιλογές - options και τις δυνατότητες που έχει το πακέτο. Όπως κάναμε ξεχωριστό wizard δηλαδή για plots και εισάγουμε κώδικα στο έγγραφο, να είναι έτσι για κάθε πακέτο. Για παράδειγμα αν ο χρήστης θέλει να εισάγει ένα block κώδικα c++ μέσα στο έγγραφο με το πακέτο listings ή minted να βλέπει εκέι τη λίστα με τις γλώσσες προγραμματισμού και τις χρωματικές επιλογές ή όποιαδήποτε επιλογή. Για το amsmath παράδειγμα να βλέπει ο χρήστης μαθηματικά περιβάλλοντα που μπορεί να εισάγει. Το ίδιο και με όλα τα άλλα πακέτα.
